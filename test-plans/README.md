# Test Plans

This folder stores executable planning artifacts for Playwright coverage.

## Canonical Location

- `test-plans/<app>/<target-slug>.md`

## Minimum Metadata

- `plan_id`
- `target_slug`
- `app`
- `plan_scope`
- `plan_type`
- `business_goal`
- `user_value`
- `confidence_target`
- `status`

## Purpose

Plans define business scenarios, risks, and expected confidence signals before automation changes are generated.
