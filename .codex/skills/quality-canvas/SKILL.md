# Quality Canvas

Create a lightweight four-quadrant artifact under `quality/plans/<app>/<target-slug>-quality-canvas.md`.

## Quadrants

1. Business Goal
2. User Value
3. Main Risks / Unacceptable Outcomes
4. Confidence Signals

## Rules

- keep it concise and reusable
- label assumptions clearly
- do not generate tests in this step

## Next Step

Route to `willenium-consultant` or `willenium-automation` after canvas approval.
