# Willenium Consultant

Use this skill to improve decision quality before execution.

## Mission

- keep automation aligned to business outcomes
- prevent false confidence from shallow checks
- route work to the right next skill

## Routing

- `quality-canvas` when input is still strategic
- `willenium-coach` when user needs scope/prompt guidance
- `willenium-test` for single bug reproduction and evidence
- `willenium-automation` for Playwright execution

## Core Review Questions

- what business objective is protected
- what user outcome matters most
- what failure is unacceptable
- what confidence decision should this work support

## Output Structure

1. Business Intent
2. Journey Value
3. Main Risks
4. Coverage Judgment
5. Recommendation
6. Weaknesses / Gaps
7. Next Best Step
