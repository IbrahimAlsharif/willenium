# Framework Structure

Willenium UI is Playwright TypeScript only.

- config: `playwright.config.ts`
- runtime: `src/pw/core/runtime/...`
- data: `src/pw/core/data/...`
- helpers: `src/pw/core/helpers/...`
- fixtures: `src/pw/fixtures/...`
- tests: `src/pw/tests/...`
- flow ownership: `flows-ts/...`
